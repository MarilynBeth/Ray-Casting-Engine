# Ray-Casting-Engine
A Ray Casting Engine built from scratch with Java using a DDA algorithm,much like the classic Wolfenstein 3D game. 


## Features:
- Menu Screen
- Wall Collision
- Different Enemies with different hit points and damage
- Doors
- Can pick up and utilize items
